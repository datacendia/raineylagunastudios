6 SIGNATURE MECHANICS (v0.3 · May 2, 2026 · shipped)
-----------------------------------------------------
01 · THE LIVING MANIFESTO
     Fraunces variable font modulated live by Lima weather via Open-Meteo.
     Humidity → wght (240-520). Cloud cover → opsz (10-24).
     Wind → letter-spacing. Same paragraph, different reading per weather.
     STATUS: working end-to-end. Fetches real weather every 10 minutes.

02 · THE HYDROPRINT LAB    (scripts/hydroprint-lab.js)
     Live WebGL 3D viewer of a ceramic mug. Drop a photo OR pick a
     house-pattern preset. Swap base colour (5 ceramics). Swap finish
     (matte/satin/gloss). Orbit to inspect. "Make this object" → WhatsApp.
     Uses Three.js ES module from CDN, PBR lighting, shadow-catcher ground.
     STATUS: working. World-first mechanic on a design studio homepage.
     Next iterations:
       [ ] More geometries beyond the mug (coaster, planter, tray)
       [ ] Pattern-on-water distortion shader for true hydro realism
       [ ] Server-side render-to-PNG so orders include a preview image

03 · REVERSE COMMISSIONING (scripts/reverse-commissioning.js)
     Visitor types their business name + category, receives an
     unsolicited 1-page brief in the studio's house template:
     observation → diagnosis → 3-phase program → investment → timeline.
     Currently deterministic templating over 6 categories (café,
     restaurant, fitness, wellness, boutique, school).
     STATUS: working. Output renders on an ivory "document" surface
     to signal it's a deliverable, not UI chrome.
     Next iterations:
       [ ] Wire to an LLM (Claude or GPT-4o) for genuinely unique copy
       [ ] Read Sereno competitive signals to make the observation specific
       [ ] Save generated briefs server-side for Stuart's review queue

05 · PROOF OF FABRICATION         (scripts/proof-of-fabrication.js · verify.html)
     Every Studios object ships with a QR printed underneath leading to
     a public /verify.html?serial=XXXX page. That page shows an Ed25519-
     signed certificate: CAD hash, print params, operator, start/finish
     timestamps, workshop climate, and the manifesto's typography state
     the day of fabrication. World-first for a design studio.
     STATUS: UI shipped (homepage panel + standalone verify page).
     DEMO-0001 lives at data/proofs/DEMO-0001.json.
     Next iterations:
       [ ] Hardware-held Ed25519 signing key (Stuart's USB)
       [ ] scripts/fab-sign.mjs CLI for one-command signing per object
       [ ] Real signature verification in verify.html using @noble/ed25519
       [ ] Printable A6 cert companion PDF

06 · THE FABRICATION TWIN          (scripts/twin-wall.js · data/twins.json)
     Every object Studios makes is fabricated twice. One ships to the
     client. The other hangs on a physical wall in the workshop. The
     homepage renders a 12x4 grid (48 cells) that fills as the wall
     fills. Hover reveals client, object, date. Click jumps to the
     proof-of-fabrication record.
     STATUS: UI shipped. Wall is currently empty (1 demo twin).
     Monthly ritual starting May 2026: photograph the wall, replace
     data/twins.json with fresh entries.
     Next iterations:
       [ ] Physical wall built in the workshop (48-cell pegboard)
       [ ] Monthly photo pipeline (iPhone → Lightroom preset → crop → JSON)
       [ ] Scale: 48 fills in ~year one; rebuild to 192 cells once full
       [ ] "Visit your twin" CTA on 1-year anniversary of any shipment

     phone → private page with: order date, Lima weather that day,
     manifesto's typography that day (applied live via Fraunces axes),
     fabrication details, operator, ship date. v0.2 is a clickable demo
     with one sample record. Real implementation needs the NFC provisioning
     pipeline in the workshop.
